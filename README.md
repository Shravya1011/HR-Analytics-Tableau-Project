# HR Analytics Tableau Project

## Project Overview
This project analyzes employee data to understand attrition patterns and workforce distribution using Tableau.

## Tools Used
- Tableau Desktop
- Microsoft Excel

## Dataset
HR dataset containing 1470 employee records including age, department, job role, and attrition status.

## Dashboard Insights
- Attrition analysis by department
- Employee distribution by age group
- Attrition trend by years at company
- Attrition by job role
